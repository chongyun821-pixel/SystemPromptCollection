## 소개
웹 인공지능들 상당히 멍청하죠? 조금 똑똑한걸 흉내내게 하기 위해 이것저것 시스템 프롬프트를 끄적여보고 있습니다.
취미로 만든것인만큼 너무 기대하지는 마시고요~

## 💾 사용 방법 (Quick Start)

1. 이 저장소에서 원하는 기능의 `.[md/txt]` 프롬프트 파일을 다운로드합니다.
   * *예: `PEV-Agent.md` (계획-실행-검증 에이전트)*
2. 평소에 쓰시는 웹 AI(ChatGPT, Claude, Gemini 등)의 채팅창을 켭니다.
3. **다운로드한 파일을 채팅창에 첨부 파일로 업로드**합니다.
4. 파일과 함께 "이 지침대로 작동해줘" 또는 "계획해~ [내용]" 처럼 첫 명령을 내리면 시스템 프롬프트가 활성화됩니다!
5. 또는 퍼플렉시티 공간, 커스텀 지피티같은데 지침으로 넣으면 된다.
